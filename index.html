Development process

History of DebiOS development process

DebiOS Opentail

This is where it started. On March 29th, the first version of DebiOS, Beta 14.3.29a, was created. There was another five versions of opentail. These versions invented most basic functions and were in czech.

Beta 14.3.29a, 14.3.29b, 14.3.30, Full 14.5.24, Beta 14.8.05 & DebiOS REVOLUTION

DebiOS One Unreleased

After few years of silence, DebiOS was reborn and almost completely recoded. There were some significant changes, such as filesystem RenEngine 1.0, needed instalation and others. These versions were in czech too and weren't never released.

DebiOS 4.1.17U, 4.5.17U 

DebiOS Public beta

After another year, DebiOS was finally moved to GitHub and finally was translated to English. This phase is still in development and there were invented big features such as iRE filesystem, Automatic Update System and some other. You can see full changelog on GitHub page.
